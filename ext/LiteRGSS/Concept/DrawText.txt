Draw text ne sera pas assuré par la classe Bitmap (après discussions) mais par un équivalent du Sprite :
Text
Propriété des textes :
- Ils peuvent être affichés dans un Viewport mais ne sont pas affecté par les shaders
- Ils contiennent le texte à afficher plutôt que l'image à afficher.
- Ils chargent le font par un ID
- Ils ont une propriété d'alignement
- Ils ont une propriété de nombre de caractères affichés (max par défaut)
- Ils ont leur taille globale (width / height) en plus de leur coordonnées

Les fonts
Les fonts sont chargés à l'aide d'un module "Fonts" qui a les fonctions suivantes :
Fonts.load_font(id, filename)
Fonts.set_default_size(id, size) #Taille par défaut

Tous les Texts chargent le Font 0 s'il n'est pas demandé au chargement.
